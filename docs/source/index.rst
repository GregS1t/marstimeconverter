Welcome to MarsTimeConverter's documentation!
=============================================

MarsTimeConverter is a Python module that converts Earth UTC time
into Martian local time formats (LMST, LTST, MSD, Ls, etc.) based
on the Mars24 algorithm.

It supports multiple Mars missions using XML configuration files,
and provides tools for visualization and comparison.
